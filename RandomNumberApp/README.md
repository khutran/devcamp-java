# RandomNumberApp

Dự án Java Console đơn giản để minh họa cách sinh số ngẫu nhiên trong Java.

## Tính năng
- Sinh số ngẫu nhiên kiểu `double` trong khoảng từ 1.0 đến 100.0.
- Sinh số ngẫu nhiên kiểu `int` trong khoảng từ 1 đến 10.
- Hiển thị kết quả trực tiếp ra console.

## Cách chạy
1. Mở terminal tại thư mục này.
2. Biên dịch mã nguồn:
   ```bash
   javac RandomGenerator.java
   ```
3. Chạy ứng dụng:
   ```bash
   java RandomGenerator
   ```

## Mã nguồn chính
Mã nguồn nằm trong file `RandomGenerator.java`, sử dụng lớp `java.util.Random`.
